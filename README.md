# Python Calculator App

A simple calculator built using Python.

## Description

This project is a basic calculator program that allows users to perform simple mathematical operations such as addition, subtraction, multiplication, and division. 

## How to Run

1. Make sure Python 3 is installed on your computer.
2. Download or clone this repository.
3. Open the folder in your code editor or terminal.
4. Run the following command:

   python calculator.py

5. Follow the instructions shown in the terminal to enter numbers and choose an operation.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero errors
- Simple and easy-to-use interface

## Technologies Used

- Python 3
